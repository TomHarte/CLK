ROM files would ordinarily go here; the copyright status of these is uncertain so they have not been included in this repository.

Expected files:

To emulate an MO5, one or more of:
* mo5-v1.0.rom — the MO5 ROM, version 1; or
* mo5-v1.1.rom — the MO5 ROM, version 1.1;

To emulate an MO6, one or more of:
* mo6-v1.rom — the MO6 ROM, version 1;
* mo6-v2.rom — the MO6 ROM, version 2; or
* mo6-v3.rom — the MO6 ROM, version 3.

Plus for use of disk images:
* cd90-640.rom — the CD90-640 floppy disk drive ROM.
